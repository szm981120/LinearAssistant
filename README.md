# LinearAssistant
线性代数小助手app

本人欲做一个计算线性代数的app，计划功能如下：

- [x] 行列式计算
- [x] 行初等变换
- [x] 矩阵求逆
- [x] 矩阵的初等运算
- [x] 矩阵求秩
- [x] 向量组相关性判断
- [x] 非奇异阵求实特征值
- [ ] 判断一个二次型是否为正定或负定
- [ ] ...

Thank you!
